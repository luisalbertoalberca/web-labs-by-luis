Javascript es un lenguaje de programación que permite crear páginas web dinámicas y con interactividad. Es uno de los pilares fundamentales del desarrollo web, junto con HTML y CSS. Con JavaScript, puedes manipular el contenido de una página, responder a eventos del usuario, realizar peticiones a servidores y mucho más. Es un lenguaje versátil que se utiliza tanto en el lado del cliente (navegador) como en el lado del servidor (Node.js).
